⚙ One-time setup required

  1. Go to Google Cloud Console → Create a project.
  2. Enable the Google Drive API and Google Picker API.
  3. Create an OAuth 2.0 Client ID (Web application type). Add file:// or your local origin to Authorized origins.
  4. Create an API Key and restrict it to the Drive & Picker APIs.
  5. Paste your Client ID and API Key into the fields below, then click Save.
